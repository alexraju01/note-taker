# Full Stack Note Taker Application

A full-stack **Note-Taking Application** built with **Node.js**, **Express**, **MySQL**, and **Sequelize** that allows users to **create**, **read**, **update**, and **delete** notes through an intuitive web interface.

## Overview

This project demonstrates how to build and deploy a complete **CRUD** (Create, Read, Update, Delete) application using a Node.js and Express backend with a MySQL database for persistent data storage. The frontend is built with **HTML**, **CSS**, and **JavaScript**, and the entire application is deployed on **Render.com**.

---

## Features

- Create new notes
- View all saved notes
- Edit existing notes
- Delete notes
- Persistent storage using **MySQL** and **Sequelize ORM**
- Fully deployed and accessible online
